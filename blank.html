  • End of the Day

    The end of the day is a great time to review behavioral performance with a student. It is a teaching moment...an opportunity. Successes can be rewarded, strategies can be taught and reviewed. Students can learn to self evaluate. This program is so simple. And, if you don’t want to enter data into the computer, the data collection form creates its own graph!!

  • On Task Analysis

    This is a half hour observation that gives you a very ‘visual’ report...lots of clear, easy to understand graphs. When shown to parents, they clearly see how their child compares to average peers. When shown to the family doctor, it provides good data to support concerns. (NOTE: This observation is usually done by a social worker, psychologist, teacher consultant, etc.)

  • How Many Times

    This is the simplest of all data collection. Just move a penny from one pocket to another each time you see the target behavior. At the end of the assessment period each day, simply count the pennies moved. Record on a chart and/or enter into the computer. Either way you get a very useful report! Check it out...
